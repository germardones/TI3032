2. Inserción de Datos: Create (C)

Las bases de datos avanzadas Documentales tienen características Schema-less (esquema dinámico al leer, sin esquema estricto de pre-escritura). ¿Qué significa esto a nivel de comandos? Que si corres con éxito una inserción apuntando directamente hacia una Colección que no existe en tu estructura previamente, MongoDB la inferirá y creará dinámicamente en medio de la operación de escritura por ti.

Comando: insertOne()

Se utiliza mandatoriamente en arquitectura para forzar la inyección de exactamente un documento monolítico a la vez. Consume vía argumento directamente la semántica de la interfaz JSON de tu Backend.

> db.clientes.insertOne({
  nombre: "Marta",
  edad: 35,
  premium: true
})

Aviso y Confirmación Terminal del Motor (Output):

{
  "acknowledged" : true,
  "insertedId" : ObjectId("64eb98f3b2a24c153ab2c418")
}

Fíjate que en tu JSON entrante nunca le parametrizaste o inyectaste ninguna Primary Key tipo ID manual. El factor de alta ventaja distribuida en MongoDB es que los algoritmos de BSON calculan criptográficamente identificadores de clúster garantizados internacionalmente a prueba de colisión por IP/Fecha denominados (ObjectId). Como desarrollador solo envías estrictamente la data útil.

Comando: insertMany()

Cuando la arquitectura te pide que ingestas reportes, datos en ráfaga o inventarios masivamente (ej. importar 10,000 líneas CSV al vuelo), el modo óptimo siempre es agrupar para matar la latencia de red contra el servidor. Usas insertMany() pasándole por argumento base un componente envolvente Array Literal Múltiple de objetos [ { ... }, { ... } ].

> db.productos.insertMany([
  { nombre: "Laptop Thinkpad", precio: 990 },
  { nombre: "Mouse Logitech", wireless: true },
  { nombre: "Bolso", material: "Tela"}
])

*Nota el polimorfismo dinámico NoSQL: el Mouse detalla un campo booleano 'wireless' que no se define en absoluto en la Laptop que pide 'precio'. A MongoDB no le genera ningún quiebre en su validación estructural y consolida e indexa los tres nodos exitosamente juntos.*
